A lean, mean, computing machine for real-time geospatial analytics


This application server is a starter kit to write pure Lua APIs. It speaks HTTP and RESP

Build some fast geospatial APIs with Redis and Tile38. It's fast

It's perfect for-

  • Building fast geospatial APIs with Redis and Tile38
  • A real-time counting engine
  • A time-series application server
  • A Publish-Subscribe notigication server
  • A cache and reverse-proxy to existing http map tile services

Themes can change any of the colors or fonts used by new.css.

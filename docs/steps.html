Usage

Here's how to start using mappist-metrics server (mms) in under 5 minutes.

Prerequisites:

Before you start following this guide, you will need to install docker, brew install make,brew install tile38, and brew install redis

Docker container

A Dockerfile is provided to build a Docker image of the server:

docker build -t mms .

Once the Docker image is built, start the container with the following command:

docker run -it --rm -p 8080:8080 mms

Docker-Compose

A docker-compose.yml file is also provided to orchestrate three containers:

  • The mappist-metrics server (mms)- see above
  • Tile38 server
  • Redis server
To start the environment, simply run : 
docker-compose up -d --build
The Docker image of the mms API is built before start due to --build option.
